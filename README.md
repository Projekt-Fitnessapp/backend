# backend

### Prerequisite

Place ```.env``` file in project root folder. with the following variables:
```
PORT=[Your Port]
DB_URL=[Your DB Url]
```

### Commands

Install:
```
npm install
```

Build:
```
npm run build
```

Run:
```
npm run start
```

Run with watch:
```
npm run devstart
```

Test:
```
npm run test
```
